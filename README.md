# Boilerplate for API express

## Install

Clone this repository and run:
```
 yarn // for install dependencies
```
And
```
 yarn dev // for start api
```

## Features:

- CORS
- Sequelize
- Mongoose
- JWT
- Bcrypt
- Yup
- DotENV
- Express Async Errors
- Eslint + Prettier
- Nodemon
- Sucrase
